# マカレン数秘「入力」シート用 — B11〜B22 の数式

Apps Script に上記コードを入れたうえで、**「入力」シート**の各セルに以下の数式を入れます。

| セル | 項目     | 数式 |
|------|----------|------|
| **B11** | 誕生数Ⅰ | `=NUM_BIRTH_I(B3,B4,B5)` |
| **B12** | 誕生数Ⅱ | `=NUM_BIRTH_II(B5)` |
| **B13** | 影数     | `=NUM_KAGE(B11,B12)` |
| **B14** | 家系数   | `=NUM_LAST_TOTAL(B1)` |
| **B15** | 自我数   | `=NUM_FIRST_TOTAL(B2)` |
| **B16** | 社会数   | `=NUM_SOCIAL_FROM_PARTS(B14,B15)` |
| **B17** | 使命数   | `=NUM_MISSION(B11,B16)` |
| **B18** | 魂数     | `=NUM_SOUL_FULL(B1,B2)` |
| **B19** | 外見数   | `=NUM_LOOK_FULL(B1,B2)` |
| **B20** | 演技数   | `=NUM_STAGE(B19,B4)` |
| **B21** | 隠数     | `=NUM_INSU(B1,B2)` |
| **B22** | 核数     | `=NUM_CORE(B11,B18)` |

## 入力シートのレイアウト（前提）

- **B1** 姓  
- **B2** 名  
- **B3** 生年（数値）  
- **B4** 生月（数値）  
- **B5** 生日（数値）  
- **B6** 聞きたい事  

## 手順

1. スプレッドシートで **拡張機能 → Apps Script** を開く。
2. 既存の「名簿に保存」用コードに、`NUM_BIRTH_I` などの**数秘関数**と `RED1` がすべて入っているか確認する。
3. 「入力」シートの **B11** に `=NUM_BIRTH_I(B3,B4,B5)` を入力。
4. **B12**〜**B22** に上表の数式を順にコピーして貼る。
5. B1〜B6 に姓名・生年月日を入力すると、B11〜B22 に各構成数が自動で表示される。

姓名は**半角・全角どちらでも**可（スクリプト内で大文字に正規化して計算します）。未入力のときは 0 や空になることがあります。
