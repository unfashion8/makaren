名前をローマ字にする方法

1. 戸籍どおりの姓名をヘボン式ローマ字で書きます

フォームには、戸籍上の姓名を「ヘボン式ローマ字」で入力してください（例: 山田太郎 → YAMADA TARO）。

代表的なかな・ローマ字の対応は次のようになります（一部抜粋）。

あ行AIUEO
か行KAKIKUKEKO
さ行SASHISUSESO
た行TACHITSUTETO
は行HAHIFUHEHO
ま行MAMIMUMEMO
や行YAYUYO
ら行RARIRURERO
わ行WAWO

きゃ・しゃ などの拗音（ゃゅょ）は、次のように書きます。

  • きゃ → KYA、きゅ → KYU、きょ → KYO
  • しゃ → SHA、しゅ → SHU、しょ → SHO
  • ちゃ → CHA、ちゅ → CHU、ちょ → CHO
  • にゃ → NYA、にゅ → NYU、にょ → NYO など

2. 注意が必要な例

  • タロウ → TARO（末尾の長音「おう」は O で表記します）
  • ユウ → YUU（長音の「う」は U を重ねて表記します）
  • オオイシ → OISHI（「おお」は1つの O で表すことが多いです）
  • ジュン → JUN（「ん」は N で表記します）
  • ホンマ → HONMA（「ん」の後ろが B・M・P のときは N のままでかまいません）
  • ハットリ → HATTORI（促音「っ」は、次の子音を重ねて表記します）
  • ハッチョウ → HATCHO（「CH」の前の「っ」は T を1つだけ入れて表記します）

※ 既にパスポートなどで決まったローマ字表記がある場合は、その表記に合わせて入力していただいても構いません。

閉じる

※ お使いのブラウザ設定によっては、自動でタブを閉じられない場合があります。その際は手動でタブを閉じてください。
